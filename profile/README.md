# A11y Studio

Accessibility intelligence for every developer.

**[A11y Studio](https://marketplace.visualstudio.com/items?itemName=a11ystudio.a11y-studio)** is a VS Code extension that catches and fixes WCAG 2.2 accessibility issues in React as you code — with plain-language explanations, one-click Quick Fixes, and direct W3C Understanding links.

| Now | Roadmap |
|---|---|
| WCAG 2.2 diagnostics in VS Code | APG Pattern Intelligence |
| One-click Quick Fixes | CSS accessibility gap detection |
| W3C Understanding links | AI-powered guidance |
| React/TSX specialist | Flow Runner — runtime testing |

No accounts. No telemetry. Free forever.
